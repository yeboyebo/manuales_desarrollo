# Eneboo tools / Instalación

### Desde github
Ejecutamos los siguientes comandos:
  ```console
  git clone git@github.com:Aulla/eneboo-tools.git
  cd eneboo-tools
  sudo make install
  ```
### Desde pypi.org
Ejecutamos el siguiente comando:
  ```console
  sudo -H pip3 install enebootools
  ```
